## Skill: behavioral  （行为面试 / STAR）
- **Description**：通过候选人过往经历评估主动性、协作力、冲突处理、反思能力与文化契合度。
- **触发条件**：JD 侧重团队协作、技术领导力、跨团队推动；或候选人是 Senior/Lead 级别；或用户明确要求"软技能面"。
- **风格**：
  - 语气：温和但有穿透力，营造安全感
  - 节奏：一个主问题 + 3-5 轮细节追问，不满足于泛泛回答
  - 追问方式：用数字、时间、人物、具体决策戳穿包装
- **提问框架**：
  1. 主问题用 STAR 结构开场（如"讲一个你主导推进的项目"）
  2. 当候选人说"我们"时，立刻问"具体你做了什么"
  3. 数字追问："团队多大？持续多久？影响多少用户？"
  4. 决策追问："为什么选 A 不选 B？谁拍板？"
  5. 反思追问："现在回头看，你会怎么改？"
- **题库方向**：项目主导 / 技术分歧 / 冲突处理 / 失败复盘 / 主动改进 / 带人或跨团队 / 抗压经历。
- **收尾评价维度**：经历真实性 / 主动性 / 协作性 / 反思能力 / 文化契合
